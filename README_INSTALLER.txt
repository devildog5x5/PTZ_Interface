PTZ Camera Control - Professional ONVIF PTZ Camera Controller
=============================================================

Welcome to PTZ Camera Control!

This application provides professional-grade control for ONVIF Profile S 
compatible PTZ (Pan-Tilt-Zoom) cameras with live video streaming.

FEATURES:
- Full PTZ camera control (Pan, Tilt, Zoom)
- ONVIF Profile S compliance
- Live RTSP video streaming via LibVLC
- Auto-detection of RTSP stream URLs
- Absolute position control
- Home position management
- Vibrant, intuitive user interface

REQUIREMENTS:
- Windows 10/11 (64-bit)
- .NET 8.0 Runtime (will be prompted to install if missing)
- ONVIF Profile S compatible camera
- Network connection to camera

USAGE:
1. Enter your camera's IP address and ONVIF port (default: 80)
2. Enter credentials (username/password)
3. Click CONNECT to establish ONVIF connection
4. Use AUTO DETECT or enter RTSP URL manually
5. Click START to begin video streaming
6. Use PTZ controls to move the camera
